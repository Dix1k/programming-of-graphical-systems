Репозиторий создан для выполнения лабораторных и практических работ по курсу **"Программирование графических систем"**. Все работы выполняются на языке C++ с использованием библиотеки OpenGL.

## 🛠️ Технологический стек

<div align="center">

[![My Skills](https://skillicons.dev/icons?i=cpp&theme=light)](https://skillicons.dev)

</div>

- **Язык программирования:** C++17
- **Графическая библиотека:** OpenGL 4.6
- **Дополнительные библиотеки:**
  - GLFW/GLEW - для создания контекста OpenGL и управления окнами
  - GLM - для математических операций (векторы, матрицы)
  - Assimp - для загрузки 3D-моделей (опционально)
  - stb_image - для загрузки текстур
